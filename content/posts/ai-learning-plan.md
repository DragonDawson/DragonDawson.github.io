从 Java 后端开发转向 AI，我制定了这样一个学习路径。

## 第一阶段：Python 入门（2~3 周）

Java 转 Python 是降维打击，重点学 AI 生态工具链：

- NumPy — 矩阵运算
- Pandas — 数据处理
- Matplotlib — 可视化
- Jupyter Notebook — AI 开发标配

## 第二阶段：数学补给（3~4 周）

- 线性代数：矩阵乘法、特征值
- 概率统计：正态分布、贝叶斯
- 微积分：梯度下降

## 第三阶段：机器学习（4~6 周）

用 scikit-learn 跑通经典算法：

- 线性回归 / 逻辑回归
- 决策树 / 随机森林
- SVM、K-means
- 模型评估

## 第四阶段：深度学习（长期）

选一个方向深入，目前考虑 NLP / 大模型方向。
